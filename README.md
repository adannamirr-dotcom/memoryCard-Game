# 🎴 Demon Slayer Memory Card Game

A themed memory card game inspired by **Demon Slayer: Kimetsu no Yaiba**.  
Players must match the Hashira cards before the timer runs out and prove their worth in the Final Selection.

## 🎮 About the Game

Welcome to the Final Selection.

Test your memory by matching pairs of Hashira characters. Each wrong match comes with consequences — the crow will judge you, and time is running out.

Can you complete the challenge?

## ✨ Features

- 🎴 Interactive memory card matching gameplay
- ⏳ 60-second countdown timer
- 🔀 Randomized card positions every game
- 🔊 Sound effects for:
  - Card clicks
  - Wrong matches
  - Game events
- 🐦 Animated crow reactions with randomized insults
- 🎬 Intro animation and themed opening sequence
- 🌸 Animated Demon Slayer inspired background
- 🏆 Win and lose popups
- 💫 Card flip animations and visual effects

## 📂 Project Structure

The game creates a deck of character cards by duplicating each Hashira character and shuffling the deck randomly.

When a player selects two cards:

- If the characters match → the pair is counted.
- If they don't match → the cards flip back and the crow reacts.
- The player wins by finding all pairs before the timer reaches zero.
